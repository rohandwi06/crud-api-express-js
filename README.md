# API CRUD Mahasiswa

API sederhana untuk mengelola dan memanipulasi data mahasiswa, yang mengimplementasikan autentikasi menggunakan JWT.

## Fitur

- **Autentikasi**: Autentikasi user menggunakan JWT
- **Manajemen Mahasiswa**: Operasi CRUD untuk data mahasiswa

## Quick Start

### Prasyarat

- Node.js
- XAMPP (MySQL)
- npm
